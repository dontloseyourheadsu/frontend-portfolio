## 🚀 Getting Started

```bash
# Clone the repository
git clone https://github.com/dontloseyourheadsu/frontend-portfolio.git

# Navigate to the project directory
cd frontend-portfolio

# Install dependencies
npm install

# Start the development server
npm start
```
